# Walkers

Настольная игра-бродилка.

## Алгоритм игры
1. Игроки по очереди бросают кости.
2. В зависимости от выпавшего числа перемещают фишку по клеткам (0 - старт).
3. При попадании на "Ловушку" пропускается ход, на "Бонус" - дополнительный бросок.
4. Побеждает тот, кто первым достигнет "Финиша".

## Сборка и тестирование
Проект использует .NET 10.0. Для сборки и запуска тестов используйте `dotnet build` и `dotnet test`.
GitHub Actions настроен на автоматизацию CI/CD.
